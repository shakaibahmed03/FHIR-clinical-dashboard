# Prompts Library

A structured collection of prompts organized by development category for the **Layer Health** project.

## Structure

```
prompts/
├── planning/       # Architecture, roadmap, sprint scoping
├── backend/        # APIs, databases, services, infrastructure
├── frontend/       # UI components, styling, user flows
├── debugging/      # Bug diagnosis, error tracing, log analysis
└── reflection/     # Retrospectives, code reviews, lessons learned
```

## How to Use

- Each category has its own folder with a `*_prompts.md` file.
- When you provide a new prompt, it will be classified and added to the appropriate file.
- Prompts follow a consistent format:
  ```
  ## [Prompt Title]
  **Context:** When to use this prompt.
  **Prompt:**
  [The prompt text]
  ```
